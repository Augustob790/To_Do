## Clone this repo

1. Clone :

```sh
  $ git clone https://github.com/Augustob790/To_Do.git
```

2. Go to folder:

```sh
  $ cd to_do_list
```

3. Execute in terminal:

```sh
  $ flutter pub get && flutter run      
```
